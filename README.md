DESCRIPTION :
This project focuses on developing a logic-based recommendation system that suggests relevant items to users based on their preferences. Unlike complex machine learning models, this system utilizes a similarity matching approach to analyze user input and align it with item attributes.

The objective is to demonstrate how intelligent recommendations can be generated using fundamental Artificial Intelligence concepts such as pattern matching and preference mapping.

STEPS :
1. Designed a dataset containing items with associated attributes (genres)
2. Collected user input in the form of preferences
3. Processed and cleaned the input data
4. Applied similarity matching logic to compare user preferences with item attributes
5. Calculated a match score for each item based on relevance
6. Ranked items according to similarity scores
7. Displayed the most relevant recommendations to the user

TECHNOLOGIES:
Python
Visual Studio Code

OUTCOME:
The system successfully generates personalized recommendations by effectively matching user preferences with item attributes. This project demonstrates a strong understanding of core Artificial Intelligence concepts such as logic building, pattern recognition, and recommendation system design.

It also provides a solid foundation for developing more advanced AI-driven recommendation systems used in real-world applications.
